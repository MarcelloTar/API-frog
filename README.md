# frog API 🐸🍹

Це API жаба, яке вміє генерувати випадкові анекдоти, підбирати назви коктейлів під настрій та створювати унікальний список інгредієнтів для коктейлю на основі вашого імені.

## Ендпоінти (Endpoints) та приклади

### 1. Отримати випадковий анекдот
Повертає один випадковий текст із бази анекдотів.

- **URL:** `/anecdote`
- **Метод:** `GET`
- **Приклад відповіді:**
  ```json
  {
    "anecdote": "помер єврей. послали іншого єврея повідомити його дружину про смерть чоловіка. приходить і питає.\n-ви дова?\n-Ні\n-На що посперечаємось?"
  }

### 2. Отримати коктейлю по настрою
Повертає один випадковий текст із бази анекдотів.

- **URL:** `/shake/:mood`
- **Метод:** `GET`
- **Приймає параметри:** `cheerful, sad, evil`
- **Приклад відповіді:**
  ```json
  {
    "cocktailsByMood": "Сльози шкарпетки"
  }

### 3. Отримати інгредієнти для коктейлю по вашому імені
Повертає один випадковий текст із бази анекдотів. Працює тільки на кирилиці(українською мовою)!!

- **URL:** `/cocktail_by_name/:name`
- **Метод:** `GET`
- **Приклад відповіді:**
  ```json
  {
    "ingredients_for_shake": [
        "банан",
        "пило",
        "мед",
        "Подих північного оленя",
        "Запах мокрого асфальту в центрі міста"
    ]
  }
